# 2.1 Windows系统驱动安装

将主板连接到电脑（如图）；

![Arduino_2.3.6-a36](./media/Arduino_2.3.6-a36.png)

打开 “ 设备管理器 ” 。

![3102](./media/3102.png)

如下图所示，电脑自动安装了驱动就可以跳过此教程。

![3103](./media/3103.png)

如下图所示，电脑没有自动安装驱动，需要手动安装。

![3104](./media/3104.png)



1、下载驱动后解压。

Windows系统驱动下载：[Windows驱动](./Windows.7z)

下载成功后解压文件。

![3105](./media/3105.png)

2、打开解压后的文件夹，点击 “ SETUP ” 。

![3106](./media/3106.png)

3、然后会有一个弹窗点击 “ 是 ” 。

4、点击 “ 安装” 。

![3108](./media/3108.png)

5、安装将成功。

![3109](./media/3109.png)

![3110](./media/3110.png)

# 2.2 Mac系统驱动安装

Mac系统驱动下载：[Mac驱动](./Mac.7z)

![3118](./media/3118.png)

1、下载驱动后解压，点击后缀为 “**.pkg**” 文件。

![3112](./media/3112.png)

2、点击 “**继续**”。

![3113](./media/3113.png)

3、点击 “**安装**”。

![3114](./media/3114.png)

4、输入你的电脑用户名与密码，然后点击 “**安装软件**”，等待安装。

![3115](./media/3115.png)

5、安装完成。

![3116](./media/3116.png)
